# Changelog

## 1.0.0

- Added a `Timestamp to Date` command to convert UNIX timestamps (seconds or milliseconds) into UTC, local time, and ISO output.
- Added a `Date to Timestamp` command to convert date strings into UNIX seconds and milliseconds.
- Added a `Current Unix Timestamp` command with live-updating current UNIX time output.
- Added copy actions for each output row and full-output copy actions.
- Added publish-ready metadata, extension/command icons, and MIT license.
